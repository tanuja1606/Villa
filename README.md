# 🏡 Villa Website Project

A responsive multi-page Villa / Property Listing website built using **HTML and CSS**.  
This project showcases a modern property website layout with multiple sections like Home, Property Details, Contact, etc.

---

## 📌 Project Overview

The Villa Website is a static front-end project designed to display villa/property information in a clean and structured format.  

It includes:
- Home Page
- Property Listing Page
- Property Detail Page
- Contact Page

This project is suitable for beginners who want to practice:
- HTML page structuring
- CSS styling
- Multi-page website navigation

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- VS Code (for development)

## 🚀 How to Run the Project

1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Open `index.html`.
4. Right-click and select **Open with Live Server** (if installed).
   OR
5. Double-click `index.html` to open in browser.
